About Cereva Tech

Founded by Christine Mac-Asore, Cereva Tech is a neurotechnology initiative building intelligent systems and machines to read, analyze, and assist the brain.

We build research-focused prototypes that combine EEG acquisition hardware, machine learning, and secure web platforms for visualization and clinical-friendly reporting.

Technology

  • EEG-based acquisition with a low-noise analog front end (ADS1299/OpenBCI)
  • Edge & cloud processing for real-time classification and storage
  • Machine learning pipelines for neural state detection
  • Secure APIs & dashboard for clinicians and researchers

Research & Applications

Applications include diagnostic support, cognitive monitoring, and rehabilitation tools. Cereva Tech focuses on responsible research practices, privacy, and safety for prototypes involving human data.
